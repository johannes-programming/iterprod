Overview
--------

This project provides a faster python-only alternative to ``itertools.product``.

Installation
------------

To install ``iterprod``, you can use ``pip``.
Open your terminal and run:

.. code-block:: shell

    pip install iterprod

Features
--------

``iterprod.iterprod(*iterables: Iterable, repeat: int = 1) -> Generator``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This generator iterates through the Cartesian product of the given iterables.

``iterprod.tests.test() -> unittest.TextTestResult``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This project can be tested through its ``test`` function.

.. code-block:: python

    import iterprod.tests
    iterprod.tests.test()
